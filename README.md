# TextSteganography

A Java console application designed to conceal secret messages inside standard text files using zero-width invisible Unicode characters.

## Description

This program provides an entry-level implementation of digital steganography. It reads a cover file, hides a custom user message using invisible character mapping, writes the result to a new file, and accurately extracts the original message during the decoding process.

## Features

* Converts secret messages to raw binary format
* Encodes text using \u200B and \u200C invisible character formats
* Saves hidden output securely to a local .txt file
* Complete automatic encoding and decoding execution

## Technologies Used

* Java
* OOP Concepts
* String Handling
* Loops and Conditions
